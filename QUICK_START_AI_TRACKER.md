# Быстрый старт: Тестирование трекера через AI агента

## За 3 шага

### 1. Запустите MCP сервер
```bash
./gradlew :server:run
```

### 2. Подключите сервер в приложении
1. Запустите приложение
2. Нажмите **"MCP"** → **"+ Добавить сервер"**
3. Введите:
   - Имя: `Local Server`
   - URL: `http://localhost:8080/mcp`
4. Нажмите **"Подключить"**

### 3. Задайте вопрос агенту

**Примеры вопросов:**
- "Сколько открытых задач в Яндекс Трекере?"
- "Покажи список открытых задач"
- "Расскажи о задаче TEST-1"

## Что происходит

1. ✅ Агент определяет, что вопрос о задачах
2. ✅ Автоматически вызывает MCP инструмент `yandex_tracker`
3. ✅ Получает данные из трекера
4. ✅ Формирует ответ на основе реальных данных

## Результат

Агент ответит примерно так:

```
Согласно данным Яндекс Трекера, в очереди TEST открыто 5 задач:

1. TEST-1: Исправить баг в авторизации
2. TEST-2: Добавить новую функцию поиска
3. TEST-4: Оптимизировать запросы к БД
4. TEST-5: Добавить тесты для модуля платежей
5. TEST-7: Настроить CI/CD
```
