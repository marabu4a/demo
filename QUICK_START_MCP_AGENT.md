# Быстрый старт: Агент с MCP инструментом Яндекс Трекера

## Шаг 1: Запустите MCP сервер

В терминале:
```bash
./gradlew :server:run
```

Сервер запустится на `http://localhost:8080/mcp`

## Шаг 2: Запустите пример агента

### Вариант A: Через IntelliJ IDEA
1. Откройте `composeApp/src/jvmMain/kotlin/org/example/demo/McpAgentExample.kt`
2. ПКМ на функции `runMcpAgentExample()` → **Run 'runMcpAgentExample()'**

### Вариант B: Через терминал
```bash
./gradlew :composeApp:run --args="mcp-agent"
```

## Интеграция в приложение

Для использования в UI приложения:

1. Запустите MCP сервер
2. В приложении: кнопка "MCP" → "+ Добавить сервер"
3. Введите:
   - **Имя**: `Local Server`
   - **URL**: `http://localhost:8080/mcp`
4. Используйте `viewModel.callMcpTool()` для вызова инструментов
